Hal yang penting dipelajari dari CSS adalah interaksi, animasi dan transisi.

Interaksi di CSS digunakan untuk menambahkan style pada elemen di kondisi tertentu, misalnya:

- Mengubah bacgkround `<div>` ketika mouse melewatinya (hover).
- Menambahkan border pada `<input>` ketika input diklik (focus).
- Mengubah warna link `<a>` ketika link sudah pernah dikunjungi (visited).
- dll

Transisi di CSS digunakan untuk menambahkan efek animasi ketika ada perubahan style CSS, agar perubahan style tidak terasa kaku.

Animasi di CSS digunakan untuk membuat animasi dengan mengkombinasikan style-style CSS, misalnya:

- Membuat elemen bergerak dari kiri ke kanan atau atas ke bawah.
- Membuat loading spinner.
- Membuat animasi perubahan warna.
